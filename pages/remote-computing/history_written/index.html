We are only briefly describing the main steps in computing hardware, limiting ourself to modern computing hardware, i.e. digital computers, and focus on how they can/could be accessed and used. We heavily rely on Wikipedia: History of computing hardware for this section.

Before we start, computer history museum has an excellent timeline starting in 1937 with a relay based adder and ending in 2015 with the Apple Watch - you can for sure run doom on it.

Basics

The principal concept of modern computers was introduced by Alan Turing when he reformulated Kurt GÃ¶dels results and introduced the Turing machine.

This was the basis for the first electromechanical computers that started developing during World War II. Most notably the devices developed in Bletchley Park by Alan Turing called bombs, used to decrypt the messages from Enigma. All of these devices mainly performed logical operations on the input that was set by hand. If you wanted something else to be done, you needed to rebuild/reconfigure the hardware. These machines where not Turing-complete, i.e. not general purpose. The machines where also electromechanical (electric switches and mechanical relays.)

The machine getting the title for Turing-complete is from Konrad Zuse that developed the world's first working electromechanical programmable, fully automatic digital computer, the Z3. Already with a lot of similarities to machines we know, one example being that it had floating-point numbers.

As input for these machines punched-cards or tapes (first used for weaving) where used to set the input variables.

The first stored-program computer was the Manchester Baby and it is also said to be the first (working) machine that contains all the essential elements of what know as computer. It was still somewhat cumbersome to program but you did not need to rearrange the internal wiring.
